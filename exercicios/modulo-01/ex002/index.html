Parágrafos e quebra de linha

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo no meio do par de tags < e >

Se precisar quebrar o texto em algum lugar específico como esse,
você pode usar a tag <br>

Vamos adicionar alguns símbolos especiais:

® → Marca registrada

© → Copyright

™ → Marca comercial

€ → Euro

£ → Libra

¥ → Yen moeda japonesa

¢ → Centavos ou cent

Δ → Letra grega Delta

→ Seta é &rarr ;

Vamos adicionar alguns emojis:

🖖 🤓

Para usar emoji precisa de &#x antes do código.
